# Summariser Skill

Use this skill when asked to summarise, condense, or create a TL;DR of any content.

## Output format

Always return summaries in this structure:

```
## Summary

**One-sentence TL;DR:** <single sentence>

**Key points:**
- Point 1
- Point 2
- Point 3

**Detail:** <2–3 sentences of elaboration if needed>
```

## Guidelines

- Be ruthlessly concise.
- Preserve the most important numbers, names, and decisions.
- Do not editorialize — stick to what the source says.
- If the content is technical, keep technical terms intact.

## No additional tools required

This skill relies only on your language capabilities — no extra tools needed.
